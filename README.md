🦈 Shark Terminal
An interactive, browser-based terminal portfolio themed around sharks. Built with vanilla JavaScript and jQuery Terminal — no frameworks, no build tools, just open and run.
Features:
Rainbow ASCII art greeting powered by lolcat
Navigate directories with ls and cd
Clickable commands and directories
Syntax highlighting as you type
Tab completion for commands and directories
Session recording via URL hash with record start/stop


type help to see all commands
tech stack:
jQuery Terminal — terminal emulator library
Figlet.js — ASCII art text rendering
Isomorphic Lolcat — rainbow color effect
Vanilla JavaScript 
the project was inspired by a blog authored by freecodecamp.org titled how to create an interactive terminal portfolio
